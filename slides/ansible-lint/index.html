What is ansible-lint?

The project was originally started by @willthames, and has since been transferred to the Ansible project team. Check the guy out at https://willthames.github.io/

How do we start using it?

pip install ansible-lint

What can we lint?

Any playbook or role

Why do we need to do it?

Because one day in the near future we will be our own father. And we want to share our roles with others

Share our roles?

Yes, there is scoring system for Ansible Galaxy and ansible-lint is being used there
